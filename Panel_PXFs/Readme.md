\[sp-o\] for an opaque panel, structural / safety  
\[sp-c\] for a clear panel, structural / safety  
\[ep-o\] for an opaque panel used for an enclosure, non-structural  
\[ep-c\] for a clear panel used for an enclosure, non-structural

Frame bottom and gantry opaque panels should be 3mm (1/8") aluminum or steel.

Working enclosure clear panels should be 5mm (3/16") polycarbonate.

Frame or printed enclosure opaque panels should be 3mm (1/8") ABS.

Frame or printed enclosure clear panels should be 3mm (1/8") acrylic, preferably tinted ("smoked").
